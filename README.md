# House_PriceEstimation_Project
End-to-end machine learning project for house price prediction using Python & XGBoost.

# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices (in lakhs) using machine learning models on Indian housing data.
It includes end-to-end data preprocessing, EDA, feature engineering, model comparison, and deployment-ready pipelines.

## 🧠 Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- XGBoost Regressor ✅ (Best Model)

## 📊 Evaluation Metrics
- R² Score
- MAE
- RMSE

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- XGBoost
- Feature-engine
- Joblib

## 🚀 Pipeline Steps
1. Data Loading & Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Outlier Handling (Winsorization)  
4. Feature Engineering  
5. Model Training & Evaluation  
6. Best Model Selection  
7. Model Serialization  

## 🏆 Best Model
XGBoost achieved the highest R² score and was saved as a production-ready pipeline.
